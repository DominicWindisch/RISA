auto risa::cuda::CropImage::processor ( int  deviceID) -> void
private

The processor()-Method takes one sinogram from the queue. Via the cuFFT-Library it is transformed into frequency space for applying the filter function. After filtering the transformation is reverted via the inverse fourier transform. Finally, the filtered sinogram is pushed back into the output queue for further processing.
